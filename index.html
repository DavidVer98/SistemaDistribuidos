Practicas de git

Algunos comandos de git

  • git init
  • git status
  • git add index.html
  • git config --global user.name ="David"
  • git config --global user.email ="david123@gmail.com"
  • git commit -m "Texto sobre el commit"
  • git log index.html "Para ver el historial del archivo"
  • git show index.html "Para ver los cambios del archivo"
